Remediation Guide

The following items require attention. Commands are provided for PowerShell (run as Administrator).

[HIGH] LSA Protection (RunAsPPL)

Current: 2 | Recommended: 1 (Enabled)

LSA Protection prevents credential theft tools like Mimikatz from accessing LSASS memory.

Set-ItemProperty -Path 'HKLM:\SYSTEM\CurrentControlSet\Control\Lsa' -Name 'RunAsPPL' -Value 1 -Type DWord

[HIGH] Attack Surface Reduction (ASR) Rules

Current: No rules configured | Recommended: Multiple rules enabled

ASR rules block common attack techniques. Recommended by ACSC and Microsoft for all enterprise systems.

See: https://learn.microsoft.com/en-us/defender-endpoint/attack-surface-reduction-rules-reference

[MEDIUM] Account Lockout Threshold

Current: Never | Recommended: 5-10 attempts

Account lockout prevents brute-force password attacks. Domain-joined systems inherit this from Group Policy.

net accounts /lockoutthreshold:5 (or configure via Group Policy for domain systems)

[MEDIUM] Remote Desktop Disabled

Current: Enabled | Recommended: Disabled (unless needed)

RDP is a common attack vector for ransomware. Disable if not required, or restrict with Network Level Authentication (NLA).

Set-ItemProperty -Path 'HKLM:\SYSTEM\CurrentControlSet\Control\Terminal Server' -Name 'fDenyTSConnections' -Value 1

[MEDIUM] PowerShell Module Logging

Current: Not Set | Recommended: 1 (Enabled)

Module Logging records pipeline execution details (Event ID 4103). Complements Script Block Logging for full visibility.

New-Item -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\PowerShell\ModuleLogging' -Force; Set-ItemProperty -Path 'HKLM:\SOFTWARE\Policies\Microsoft\Windows\PowerShell\ModuleLogging' -Name 'EnableModuleLogging' -Value 1

[MEDIUM] Controlled Folder Access

Current: Disabled | Recommended: Enabled

Controlled Folder Access blocks unauthorized apps from modifying protected folders (Documents, Pictures, etc.). Key ransomware defense.

Set-MpPreference -EnableControlledFolderAccess Enabled

[MEDIUM] Print Spooler Service

Current: Running (Automatic) | Recommended: Disabled (if not printing)

Print Spooler has critical vulnerabilities (PrintNightmare). Disable on systems that don't need printing, especially servers.

Stop-Service -Name Spooler -Force; Set-Service -Name Spooler -StartupType Disabled

[LOW] Administrative Shares

Current: 3 admin shares (C$, ADMIN$, etc.) | Recommended: Disabled (if not needed)

Admin shares (C$, ADMIN$) allow remote access. Used by lateral movement tools like PsExec. Disable if not required.

Set-ItemProperty -Path 'HKLM:\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters' -Name 'AutoShareWks' -Value 0
